# Home

# You will now be greeted by the home screen, where you will find a quick overview of your

-   **Most recently used content**  
    As this already suggests, you will see the last opened or uploaded items here.
-   **Your marked as favorite files**  
    Long press any file or folder and mark it as favorite and youu will find it here.
-   **The shared area:**
    -   Shared with you
    -   Shared with others  
        Long press on any file and folder and select the contact you want to share the item to.  
        If someone shared something to you, you will also see it here.
-   **The marked as offline available content**  
    If you ever need your data without an internet connection, you can also make the relevant items available offline by pressing and holding.
-   **The trashed items**  
    Items that you delete will be placed in the trash for 30 days, where they can be restored. Manual deletion is also possible, of course.
